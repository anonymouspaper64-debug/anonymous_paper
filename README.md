# anonymous_paper files 

This repo contains datasets for logic proofs extracted from logic tutoring systems, as well as the prompts used to generate the single-step student-LLM and corresponding feedback interactions.

The prompts include: 
  - Tutor prompt 
  - Teacher prompt 
  - Judge prompt 

  The KG files represent the construction and traversal of KG 

  The num_* map_* and prop_* files are used to construct the KG
